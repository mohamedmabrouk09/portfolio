01 — About Me

Not just another
engineer.

I'm a software engineering student at ENSA Khouribga currently navigating the cycle d'ingénieur on a strict diet of caffeine, curiosity, and a deeply unreasonable number of side projects.

It started with movies, books, and the audacity of imagining the future. Tony Stark building things in a cave. Sci-fi cities that felt just plausible enough to chase. I didn't just want to watch that future. I wanted to build it. So I opened a code editor. And I never really closed it.

I gravitate toward clean architecture, scalable systems, and developer tools that make other developers' lives easier. If there's a tedious setup process somewhere, I've probably already built a CLI to kill it.

When I'm not in the terminal, I'm watching Real Madrid surgically dismantle someone's evening, or running coordinated flanks in Rainbow Six Siege — which has, ironically, taught me more about decision-making under pressure than any classroom ever did.
